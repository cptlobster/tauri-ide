# tauri-ide

work in progress

## Prerequisites

Follow [Tauri's prerequisite guide](https://v2.tauri.app/start/prerequisites/) for your platform of choice to get started. You'll need to install the native webview, Rust command line tools, and NodeJS.

## Running

```sh
npm run tauri dev
```
